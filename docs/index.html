    • Metamon

    Share Data Representations (Models, Collections) between BinarySearchTree and the Browser

    Read the DOCS

    Why?

    Metamon is a toolkit for consistency with the representations of your data between Browsers and BinarySearchTree Servers. It provides methods for creating these representations and for indexing these representations.

    Installation

    Metamon is available via npm for BinarySearchTree, and bower for the Browser

    BinarySearchTree

    npm i -S metamon

    Browser

    Metamon has 3 bundles for your convenience:

    Minified

    const {Model, Collection, Enum} = require('metamon/dist/metamon.min.js');

    Un-minified

    const {Model, Collection, Enum} = require('metamon/dist/metamon.js');

    CommonJS

    const {Model, Collection, Enum} = require('metamon/dist/metamon.cjs.js');

    You can install metamon via bower with:

    bower install metamon
